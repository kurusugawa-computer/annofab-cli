==========================================
organization_member delete
==========================================

Description
=================================
組織からユーザを脱退させます。


Examples
=================================


基本的な使い方
--------------------------

対象組織の名前を ``--organization`` に、脱退させるユーザのuser_idを ``--user_id`` に指定します。


.. code-block::

    $ annofabcli organization_member delete --organization org1 --user_id u1 u2
