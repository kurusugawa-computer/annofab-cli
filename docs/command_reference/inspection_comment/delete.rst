==========================================
inspection_comment delete
==========================================

Description
=================================
検査コメントを削除します。


.. warning::

    非推奨なコマンドです。2022/12/01以降に廃止する予定です。
    替わりに `annofabcli comment delete <../comment/delete.html>`_ コマンドを使用してください。




Examples
=================================


`annofabcli comment delete <../comment/delete.html>`_ コマンドを参照ください。
コマンドライン引数の使い方は同じです。
