annotation_zip list_bounding_box_annotation
====================================================================================


Description
=================================
アノテーションZIPからバウンディングボックス（矩形）アノテーションの座標情報を出力します。

Examples
=================================

.. code-block:: bash


    $ annofabcli annotation_zip list_bounding_box_annotation --project_id prj1 --output out.json --format pretty_json
